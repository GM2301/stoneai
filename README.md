# StoneAI — Universal Stone Intelligence

Sistemi i parë universal i Quality Control për çdo lloj guri natyror dhe artificial.
Funksionon pa katalog të caktuar — njeh çdo mermer, granit, gëlqeror, travertin, kuarcit, onyx dhe më shumë.

---

## SETUP

```bash
npm install
```

Krijo `.env.local`:
```
OPENAI_API_KEY=sk-proj-...
```

```bash
npm run dev        # http://localhost:3000
npx vercel         # Deploy
```
